**if-else statement:
*syntax:

if [ condition ]
then
true statements
else
false statements
fi


**if..elif..else..fi statement:
*syntax:

if [ expression1 ]
then
   statement1
   statement2
   .
   .
elif [ expression2 ]
then
   statement3
   statement4
   .
   .
else
   statement5

*Example:

#!/bin/bash

os=`cat /etc/os-release|grep ^NAME|awk -F '"' '{print $2}'|awk '{print $1}'`
if [ $os = Amazon ] || [ $os = Redhat ] || [ $os = Centos ]
then
echo My distribution is $os
elif [ $os = Windows ]
then
echo My distribution is $os
elif [ $os = ubuntu ]
then
echo My distribution is $os
else
echo My distribution is not $os
fi
